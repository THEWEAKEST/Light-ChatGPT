# Light-ChatGPT

> A local chat bot

> Based on Element-plus Flask and MySQL

## Project setup

```bash
npm install
```

### Compiles and hot-reloads for development

```bash
npm run dev
```

### Compiles and minifies for production

```bash
npm run build
```

## Usage

```bash
git clone https://github.com/THEWEAKEST/Light-ChatGPT
cd Light-ChatGPT
npm i
npm run dev
```

### Custom theme

See `src/styles/element/index.scss`.
